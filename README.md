# Influencer Marketing ROI Dashboard

A data analytics dashboard designed to analyze influencer marketing performance across platforms, campaign types, and audience segments.
This project focuses on business intelligence, KPI tracking, and data storytelling using interactive visualizations.

---

## 📊 Overview

This dashboard analyzes 1,826 influencer marketing campaigns across 4 major platforms from 2022 to 2026, covering over 150,000 cleaned records.

It is designed to simulate a real-world marketing analytics system used for performance tracking, ROI measurement, and strategic decision-making.

---

## Key Metrics

* Total Campaigns: 1,826
* Total Revenue: $4.52M
* Total Reach: 917.8M
* Total Engagements: 88.4M
* Average Engagement Rate: 31.2%

---

## 🔑 Key Insights

* Instagram is the highest revenue-generating platform
* Twitter delivers the highest engagement rate among all platforms
* Food category contributes the highest overall revenue
* Brand Awareness campaigns perform best in ROI
* Wednesday is the strongest performing launch day

---

## 📈 Features

* Interactive KPI dashboard with real-time visual structure
* Revenue trend analysis across months and years
* Platform-wise performance comparison
* Influencer category segmentation analysis
* Campaign type performance evaluation
* Audience engagement behavior tracking
* Heatmap analysis for platform vs category performance
* Multi-tab dashboard navigation system
